# DrillSense AI

## GPU-Accelerated Decision Intelligence for Oilfield Monitoring

DrillSense AI is an AI-powered decision intelligence platform that helps petroleum engineers identify high-risk wells, detect anomalies, prioritize maintenance, and generate engineering recommendations using machine learning and Google's Gemini AI.

## Problem

Oilfield engineers monitor thousands of sensor readings every day. Manual monitoring delays decision-making and increases operational risk.

DrillSense AI automatically detects anomalies, predicts operational risks, ranks wells by priority, and generates AI-assisted engineering reports for faster decision-making.


## Features

- Synthetic industrial telemetry dataset (432,000 records)
- Feature Engineering Pipeline
- Isolation Forest anomaly detection
- XGBoost risk classification
- Gemini-powered engineering reports
- Executive Dashboard
- Well Explorer
- AI Decision Center
- GPU Performance Benchmark (RAPIDS cuDF)
- Google Cloud Storage integration
- Google BigQuery integration
